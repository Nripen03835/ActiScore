ActiScore API Documentation

Overview

The ActiScore API provides programmatic access to emotion analysis functionality. You can analyze videos for facial emotions, audio for speech emotions, or both for multimodal analysis.

All API endpoints require authentication. You can authenticate using your ActiScore account credentials.

Endpoints

Video Analysis

POST /api/v1/analyze/video
Content-Type: multipart/form-data

file: [video file]

Audio Analysis

POST /api/v1/analyze/audio
Content-Type: multipart/form-data

file: [audio file]

Multimodal Analysis

POST /api/v1/analyze/fusion
Content-Type: multipart/form-data

video: [video file]
audio: [audio file]

Get User's Analyses

GET /api/v1/analyses

Get Analysis Details

GET /api/v1/analyses/{id}

Response Format

All API responses are in JSON format and include a success field indicating whether the request was successful.

Example Response

{
  "success": true,
  "analysis_id": 123,
  "results": {
    "emotions": {
      "happy": 0.7,
      "sad": 0.05,
      "angry": 0.1,
      "surprised": 0.1,
      "neutral": 0.05
    }
  }
}
